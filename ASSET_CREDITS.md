# Asset Credits

이 프로젝트의 기본 빌드에는 외부 그래픽 에셋 파일이 포함되지 않습니다.

- 고등어 직원, 몬스터, 유물, 카트, 호스, 시설 바닥, 조명, 파티클: Phaser Graphics/Canvas로 런타임 절차 생성
- PWA 아이콘: 프로젝트 내 직접 제작 SVG
- UI 아이콘: 유니코드 기호와 CSS 도형

따라서 기본 빌드의 그래픽에는 별도 저작자 표시가 필요한 CC-BY 자료가 없습니다. 향후 외부 에셋을 추가할 때는 파일명, 원본 URL, 제작자, 라이선스, 수정 내역을 이 문서에 기록해야 합니다.
